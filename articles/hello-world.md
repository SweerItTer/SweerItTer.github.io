## 关于本博客

这是一个基于 GitHub Pages 的静态博客网站，具有以下特点：

- 美观的欢迎页设计
- 单页应用（SPA）架构
- 支持 Markdown 和 HTML 文章格式
- 代码高亮功能
- 目录导航
- 搜索和排序功能

## 快速开始

### 添加新文章

1. 在 `articles` 文件夹中创建新的 Markdown 或 HTML 文件
2. 在 `articles/articles.json` 中添加文章信息
3. 提交并推送到 GitHub

## 代码高亮测试

### Python 示例

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
